xG Efficiency Lab

Does Barcelona score as much as the quality of its chances says it should? These charts compare expected goals (xG) with what actually happened.

New here? What is xG?

Expected goals (xG) rates every shot by how likely it was to be scored, from 0 to 1. A tap-in from the six-yard box might be 0.7 xG; a hopeful shot from 30 metres might be 0.03 xG. Add up the shots and you get how many goals an average finisher would have scored.

So if Barcelona racks up 2.5 xG but scores 1, they created good chances and wasted them. Score 3 from 1.0 xG and they finished brilliantly — or got lucky.

Where does the xG come from? We built the model ourselves. Every shot is scored by a logistic-regression model trained on 11,830 real shots — the whole 2025/26 La Liga season plus the 2026 World Cup — against what actually happened to them. It weighs the shot's distance, the angle the goal mouth opens up, whether it was a header, a big chance, and the situation (open play, free kick, corner, set piece, fast break), with a La Liga finishing calibration (penalties fixed at 0.76). At match level it tracks Understat's professional model at r = 0.975 on Barcelona's games. The same model runs across our La Liga, World Cup and Barcelona dashboards, so the numbers compare cleanly.

Action maps

The Shots map is a half pitch with the goal at the top — dots sized by xG: green = goal, filled red = on target, hollow red = off target / blocked. Take-ons, tackles & passes use the full pitch (attacking →) with green = success, red = fail arrows. Each map lists its totals with the success / conversion split. Hover any mark for the minute, xG and the game.

Style fingerprint

Each spoke is the player's percentile rank among squad players with 120+ minutes, measured per 90 minutes (Rating is the raw average). Further out = higher in the squad; 100 = the best. Comparing overlays a second player. Hover a point for the exact value.

What each spoke means
  • Finishing — goals per 90
  • G+A — goals + assists per 90
  • Shooting — shots per 90
  • Creativity — key passes per 90 — passes that directly set up a shot (the last pass before a teammate's attempt), per WhoScored
  • Dribbling — take-ons won per 90
  • Defending — tackles + interceptions per 90
  • Aerials — aerial duels won per 90
  • Rating — average WhoScored match rating (percentile of the raw value)

Data

The dashboard is built from cached WhoScored match feeds plus Understat xG. Everything here is generated by the project pipeline, which re-scrapes and rebuilds roughly one hour after each match.

How it updates

A scheduled job runs about one hour after every Barcelona kickoff: it scrapes the finished match, recomputes player & xG aggregates, regenerates this dashboard and the per-match pages, then publishes to GitHub Pages.
